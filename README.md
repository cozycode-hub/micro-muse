# MicroMuse ✦

AI-powered content idea generator for creators.

Pick a niche. Get ideas. Generate scripts, captions, and posts for TikTok, YouTube, Substack, Instagram, LinkedIn, and Podcast — in seconds.

## Live App
https://micromuse-eta.vercel.app

## Built With
- React + Vite
- Claude API (coming in v2)
- Deployed on Vercel

## Run Locally
git clone https://github.com/cozycode-hub/micro-muse.git
cd micro-muse
npm install
npm run dev
